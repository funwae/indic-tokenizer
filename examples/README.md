# Examples

This directory contains example scripts demonstrating how to use the Indic Tokenization Lab.

## Basic Comparison

`basic_comparison.py` - Simple example of comparing tokenizers on a text.

```bash
python examples/basic_comparison.py
```

## Evaluation

`evaluate_tokenizer.py` - Example of evaluating a tokenizer with comprehensive metrics.

```bash
python examples/evaluate_tokenizer.py
```

## More Examples

Additional examples will be added as the project develops:
- Training tokenizers
- Batch evaluation
- Scorecard generation
- Custom tokenizer integration
