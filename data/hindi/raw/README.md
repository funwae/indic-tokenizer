# Raw Hindi Corpus

This directory should contain the raw Hindi corpus files.

## CC-100 Hindi

Download from: https://data.statmt.org/cc-100/hi.txt.xz

After decompression, place `hi.txt` here.

## IndicNLP Hindi (Alternative)

Download from: https://objectstore.e2enetworks.net/ai4b-public-nlu-nlg/v1-indiccorp/hi.txt

Place as `indicnlp_hi.txt` here.

See `docs/31-corpus-prep-hi.md` for detailed instructions.
