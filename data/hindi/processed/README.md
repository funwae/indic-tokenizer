# Processed Hindi Corpus

This directory contains processed corpus files ready for tokenizer training.

## Files

- `gpe_cbpe_hi_corpus.txt`: Processed corpus (500k lines) for GPE+CBPE and AG-BPE training
  - Generated by: `scripts/prepare_corpus_hi.py`
  - Size: ~110MB
  - Format: One sentence per line, UTF-8, Devanagari text

These files are generated locally and not committed to git (see .gitignore).
