# Scorecards

This directory contains evaluation scorecards generated by the evaluation scripts.

Scorecards are generated in Markdown format and contain:
- Comprehensive metrics (fertility, chars/token, grapheme violations)
- Comparison across multiple tokenizers
- Timestamp and evaluation details

## Generating Scorecards

```bash
# Single evaluation
python scripts/evaluate_tokenizers.py \
    --text "यहाँ आपका हिंदी वाक्य जाएगा।" \
    --tokenizers all \
    --output scorecards/single_eval.md

# Full evaluation on all datasets
python scripts/run_full_evaluation.py \
    --output-dir scorecards
```

## Scorecard Format

Scorecards include:
- Tokenizer name and ID
- Metrics (fertility, violation rate, etc.)
- Sample texts used for evaluation
- Timestamp
